# Web Security Labs

## Overview

This section documents practical web application security assessments performed in a controlled laboratory environment.

The objective is to understand common web application vulnerabilities, apply secure testing methodologies, and recommend defensive measures based on industry best practices.

All testing was conducted against intentionally vulnerable applications or systems where explicit authorization was granted.

---

## Objectives

- Understand web application architecture
- Perform reconnaissance against web applications
- Identify common vulnerabilities
- Practice secure testing techniques
- Document findings professionally
- Recommend mitigations

---

## Skills Demonstrated

- HTTP Enumeration
- Web Reconnaissance
- Directory Enumeration
- Authentication Testing
- Session Analysis
- Cookie Security
- Burp Suite
- OWASP ZAP
- OWASP Top 10
- Secure Reporting
